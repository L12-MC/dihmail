dihmail

dihmail is an experimental, encrypted messaging system featuring per‑message symmetric AES‑256‑GCM encryption with a wrapped key (envelope encryption) design, rich text support, and encrypted attachments. It explores alternative addressing via random local parts (alias:dih:domain) and optional multi‑platform interfaces (web + desktop).

This project is in active evolution & not intended for production use. Expect breaking changes and security refinements.
